# 8BitScraper
8BitScraper is a retro video game box art web scraper built in Python.

# Demo
![pic](https://imgur.com/a/O0cMIZB)

# Usage

Run ``` python scraper.py <search> <filename>``` in the terminal or CMD on Windows
